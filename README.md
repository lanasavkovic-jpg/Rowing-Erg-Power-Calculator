# Rowing-Erg-Power-Calculator
Turns out the rumours are true... rowing does depend on height :(

This program allows you to analyze wattage produced on the rowing machine depending on height and initial applied force. Equations were derived with a dynamic analysis of the rowing machine using concepts learned in MAAE2101C (dynamics 2). Simplifications and assumptions were made.

To run this program, download 'Rowing Erg Power Calculator.py' and run it on a Python IDE such as Wing 101 or vs code.
